Declarative Programming languages and techniques focus on

  • Specifying what you want...
  • ...not how to get it

Logic Programming

A subparadigm of declarative programming where we get results through application of rules about a problem domain

The language manages flow of control by applying the rules across its data structures and constructing results

Goodbye, IF

Someone else has implemented flow of control in a single place and I can focus on defining what I want my program to do

PuzzleScript

Game engine using a rules based DSL to define game logic

  • Implemented in JS - create and publish games from the web
  • Lives here: http://www.puzzlescript.net/
  • Created by @increpare

It's Logical; Be Declarative

Higher level abstractions and different approaches help us focus on solving problems

Lower level artifacts like IF are important but too widely present in our code
